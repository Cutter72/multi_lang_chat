# Multi lang chat
### PL
Komunikator na system Android oraz iOS pozwalający na tłumaczenie wiadomości na dowolny język.
### EN
A messenger for Android and iOS that allows you to translate messages into any language.
### Technologie/Technologies:
* Android, iOS
* Dart
* Flutter (Material Design 3)
* Futures, Streams
* Firestore
* Firebase Authentication
* Crashlytics
* Dart Mappable (freezed alternative)
